# Prompt — Área do mestre e campanha viva

Você está evoluindo o sistema da campanha Ghosts of Saltmarsh.

Objetivo:
Criar recursos para o mestre controlar a mesa.

Recursos desejados:
- Painel do mestre.
- Lista de personagens.
- Vida, nível e corrupção em visão rápida.
- Facção de cada personagem.
- Pistas já descobertas.
- Segredos ligados ao Thalassyr.
- Registro de sessões.
- Registro de NPCs.
- Controle de corrupção por personagem.
- Campo de notas privadas do mestre.

Regras:
- Jogador não deve ver notas privadas do mestre.
- Cifra das Marés é segredo/mecânica, não nome da campanha.
- Corrupção é a barra narrativa principal.
- Não remover a ficha atual.
- Manter mobile-first.
