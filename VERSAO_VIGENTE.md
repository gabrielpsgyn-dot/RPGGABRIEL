# RPG Gabriel — Versão Vigente Única

Este pacote consolida o projeto em uma única versão vigente.

## O que foi limpo

- Removidos arquivos soltos antigos de auditoria versionada.
- Removidos comandos antigos `COMANDOS_ATUALIZAR_Vxx`.
- Removidos blocos inline de tema V48, V49, V50, V51, V52, V53, V54 e V55.
- Criado um único tema oficial:
  `public/css/tema-vigente.css`
- As páginas passam a usar `?v=vigente`.

## Páginas principais

- `public/index.html`
- `public/mestre.html`
- `public/livro-regras.html`
- `public/criador.html`

## Regra a partir daqui

Não criar nova cascata de arquivos `Vxx` para cada ajuste visual.

Quando houver correção, atualizar a versão vigente.
