# V37 — Mobile: uso real

## Correções

- Removido o painel de passo a passo da tela da ficha.
- A ficha não deve virar tutorial gigante.
- Ataques agora usam lista de armas.
- Ataques cadastrados aparecem como cards no celular.
- Tabela de ataque fica apenas como apoio em telas maiores.
- Inventário ganhou botão direto para sincronizar recebidos.
- Adicionar item manual ficou recolhido.
- Inventário preserva objetos com grantKey para não quebrar sincronização.

## Fluxo de uso no celular

1. O jogador escolhe classe/raça/antecedente.
2. Toca em "Sincronizar recebidos" no inventário se necessário.
3. O jogador adiciona ataque por lista de armas.
4. Durante o jogo usa cards: rolar ataque, rolar dano, remover.

## Pendências futuras
