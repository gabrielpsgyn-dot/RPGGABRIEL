# V44 — Equipamentos iniciais para todas as classes

## Correção

A V43 corrigia o Mago, mas o problema é sistêmico. Agora existe uma tabela de equipamento inicial para todas as classes.

## Regra

Cada classe pode ter:

- item fixo;
- escolha que cria item;
- escolha narrativa;
- pacote inicial;
- foco;
- armadura;
- ferramenta;
- arma.

## Classes contempladas

- Bárbaro
- Bardo
- Bruxo
- Clérigo
- Druida
- Feiticeiro
- Guerreiro
- Ladino
- Mago
- Monge
- Paladino
- Patrulheiro

## Observação importante

Os itens são templates editáveis. O conteúdo exato deve ser ajustado conforme o livro do Gabriel.
O sistema agora suporta todas as classes; não é mais um remendo só para Mago.
